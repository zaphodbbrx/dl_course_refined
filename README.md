# Курс «Deep Learning»

Курс по глубокому обучению для магистратуры 1 курса. Лекции охватывают архитектуру Transformer, механизмы внимания, обучение и инференс языковых моделей.

## Структура

```
├── lec01/ — ...
├── lec02/ — ...
├── ...
├── lec06/ — Архитектура Transformer
├── lec07/ — Языковые модели (BERT, GPT, RLHF, DeepSeek, MoE)
├── images/ — иллюстрации
└── transformer_presentation.py — генерация слайдов Transformer
```

Каждая лекция включает:
- `lecXX.tex` — конспект (article)
- `lecXX.pdf` — конспект в PDF
- `lecXX_slides.tex` — слайды (beamer)
- `lecXX_slides.pdf` — слайды в PDF
